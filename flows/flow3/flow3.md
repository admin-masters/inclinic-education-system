# Field Rep Flow

Recruitment Link -> Field Rep Gmail Login  
Field Rep Gmail Login -> Authentication -> Welcome  

New User Flow:

Welcome -> Field Rep Registration  
Field Rep Registration -> Enter Details  
Enter Details -> Set Password  
Set Password -> Security Question  
Security Question -> Register  
Register -> Login  

Existing User Flow:

Welcome -> Share Collaterals Page  

Share Flow:

Share Collaterals Page -> Select Doctor  
Select Doctor -> Enter WhatsApp Number  
Enter WhatsApp Number -> Select Collateral  
Select Collateral -> Submit  
Submit -> Generate WhatsApp Message  
Generate WhatsApp Message -> Open WhatsApp  
Open WhatsApp -> Send Message  
Send Message -> Status Updated  

Alternative:

Search Doctor -> Send Message -> Sent
