# Publisher Campaign Flow

Publisher Login -> Publisher Home -> Add Campaign  
Add Campaign -> Generate Campaign ID -> Select Systems  
Select Systems -> Continue -> Create Campaign Step1  
Create Campaign Step1 -> Enter Company Details -> Enter Brand Details  
Enter Brand Details -> Enter Doctor Count -> Enter Contact Details  
Enter Contact Details -> Upload Banners -> Save  
Save -> Campaign Step1 Saved -> Choose System  
Choose System -> Open InClinic System -> Publisher Landing Page  
Publisher Landing Page -> Add Campaign Details -> Create Campaign Details  
Create Campaign Details -> Upload Assets -> Submit  
Submit -> Update Campaign Details -> Review  
Review -> Campaign Details Display -> Complete  

Alternative Flow:

Publisher Home -> Edit Campaign -> Update Campaign Details -> Save -> Display
