# Doctor Collateral Access Flow

WhatsApp Message -> Short Link Click  
Short Link Click -> Verification Page  
Verification Page -> Enter WhatsApp Number  
Enter WhatsApp Number -> Verify Access  
Verify Access -> Authentication Success  

Authentication Success -> Unlock Content  
Unlock Content -> View Collateral  
View Collateral -> Scroll Content  
Scroll Content -> Play Video  
Play Video -> Download PDF  
Download PDF -> Browse Archives  

Alternative:

Invalid Number -> Verification Failed  
Verification Failed -> Reenter Number -> Verify Access
